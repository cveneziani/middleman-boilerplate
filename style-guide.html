Frameworks

Middleman Boilerplate is based on ZURB Foundation framework. We use their grid system, typography, buttons, forms and so on. Please refer to their documentation for anything that is not specified in this guide.

It's also based on Compass

Base

Typography

We use several fonts for different purposes:

  • Open Sans for forms
  • François One for headers
  • Lato for any other text

Forms

We use Open Sans font for forms.

Headers

We use François One font for headers. We've defined up to 6 levels but we mainly use only the first 3 ones.

It will apply the heading style to the span.
